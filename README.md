# Customer-Analytics-Dashboard

PROJECT OVERVIEW

This project integrates multiple datasets including:
• Customer Master Data
• Transactions Data
• Subscription Data
• Engagement Data

PROJECT OBJECTIVE

   To develop an interactive Power BI dashboard that analyzes customer behavior, churn, and revenue to support data-driven decision-making.

TOOLS USED

    Power BI, DAX, Visual Charts, Date Table, Slicers, Drill_through

DESCRIPTION OF DASHBOARD

  Customer Overview: Summary of customers, revenue, and demographics
  Customer Segmentation: Customer distribution by region, type, and plans
  Engagement Analysis: Insights on transactions and customer activity
  Churn & Retention: Analysis of churn trends and retention rates
  Revenue & Performance: Overview of revenue trends and performance

KEY INSIGHTS

    Customer engagement and subscription status strongly influence revenue and retention. Higher activity customers generate more value, while churn is concentrated in specific regions and plans. Focusing on high-performing segments and reducing churn can significantly improve overall performance.
